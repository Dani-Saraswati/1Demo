 git commands:
    To check status-> git status
    to add file->  git add ""
    to add all changes-> git add .
    to commit it-> git commit -m "message"
    to push to github-> git push origin main

    To check branch-> git branch
    to rename branch ->  git branch -M main
    to navigate->  git checkout -branch name-
    to create new branch->  git checkout -b -new branch name-
    to delete branch-> git branch -d -branch name-

    git pull origin main-> to get changes made on git to local
